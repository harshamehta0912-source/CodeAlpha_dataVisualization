# Sample Superstore Data Visualization

## Internship Project - CodeAlpha

## Acknowledgement
This project was completed during my internship at CodeAlpha.

---

## About the Project
I worked on the Sample Superstore dataset to understand sales, profit, and customer trends. The idea was to turn raw data into simple visuals so that business insights can be easily understood.

---

## Tools Used
- Power BI
- Microsoft Excel

---

## What I Built
I created an interactive dashboard with:
- Sales and Profit overview
- Region and State-wise analysis
- Category and Sub-category performance
- Monthly sales trend
- Top products analysis
- Discount vs Profit comparison

---

## Key Learnings from Data
- Technology products gave the highest sales.
- High discounts reduced profit in many cases.
- Different regions showed different performance levels.
- Office Supplies stayed consistent in sales.
